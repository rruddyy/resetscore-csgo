
# Reset Score

A sourcemod plugin which allow player to reset his/her score, with random global messages.

## Commands

To reset score:- `!rs` or `sm_rs`

Note: commands starting with `!` are chat commands



```
## Installation

To compile this plugin you can use https://spider.limetech.io

1.  Put `resetscore.smx` to `addons/sourcemod/plugins` directory
2.  Restart Server & Done
